## :bangbang: Folder Structure

Here is the folder structure of this app.

```bash
Next15-LandingPage/
  |- components/
  |- app/
    |-- fonts/
    |-- global.css
    |-- app.tsx
    |-- layout.tsx
  |- public/
  |- .eslintrc.json
  |- .gitignore
  |- next.config.js
  |- package-lock.json
  |- package.json
  |- postcss.config.js
  |- tailwind.config.js
  |- tsconfig.js
```

## :toolbox: Getting Started

1. Make sure **Git** and **NodeJS** is installed.

2. Clone this repository to your local computer.

3. Open terminal in root directory. Run `npm install --legacy-peer-deps` or `yarn install --legacy-peer-deps`.

4. Now app is fully configured üëç and you can start using this app using `npm run dev` or `yarn dev`.
